##Задачи

1.
Создать класс «Рациональная дробь» и реализовать основные методы. Написать функцию вычисления длины периода рациональной дроби  .

2.
Создать класс Integer для длинных целых чисел. Реализовать основные операции: вывод-вывод, сложение, умножение. Для хранения цифр использовать динамический массив или связный список.

3.
Создать класс Date для вывода текущей даты в виде dd.mm.gg. Используя этот класс, найти количество дней между заданными датами. Учесть високосные годы и т. д.

4.
Создать класс «Многочлен» для хранения данных в виде связного списка  .
Если какие-либо целочисленные коэффициенты равны нулю, соответствующие узлы пропускаются. Перегрузить операции =, сложения, умножения многочленов, нахождения производной, вывода. По схеме Горнера найти значение многочлена в точке.

5.
Создать класс «Квадратная матрица»  -го порядка. Определить конструкторы, перегрузить операции ввода/вывода. Вычислить определитель, используя рекурсивную функцию  , где   – матрица, полученная удалением 1 й строки и  -го столбца.

6.
Создать класс Date для работы с датами в формате «год, месяц, день». Класс должен включать не менее трех конструкторов инициализации: числами, строкой вида «год, месяц, день» (например, «2009.09.01») и датой. Обязательными операциями являются: вычисление даты через заданное количество дней, вычитание заданного количества дней из даты, определение високосности года, присвоение и получение отдельных частей (год, месяц, день), сравнение дат (равно, до, после), вычисление количества дней между датами.

7.
Создать класс Triad (тройка чисел); определить с помощью перегрузки операции ++ операцию увеличения всех полей одновременно на 1. Определить производный класс Date (дата) с полями: год, месяц и день. Переопределить операции (++, --) увеличения (уменьшения) даты на один день с учетом високосного года и определить операцию увеличения даты на n дней.

8.
Создать класс Decimal как производный от класса Array (динамический массив). Класс Decimal для работы с беззнаковыми целыми десятичными числами использует для представления числа динамический массив типа char, каждый из элементов которого содержит десятичную цифру. Младшая цифра имеет меньший индекс. Перегрузить арифметические операции (+, -, *, /) аналогичные встроенным для целых чисел, и операции сравнения (==, !=, <, >).

9.
Создать класс Hex как производный от класса Array (динамический массив). Класс Hex для работы с беззнаковыми целыми шестнадцатеричными числами использует для представления числа динамический массив типа char, каждый из элементов которого содержит шестнадцатеричную цифру. Младшая цифра имеет меньший индекс. Перегрузить арифметические операции (+, -, *, /) аналогичные встроенным для целых чисел, и операции сравнения (==, !=, <, >).

10.
Создать класс Octal (длинные беззнаковые восьмеричные) как производный от класса Array (динамический массив). Класс Octal для работы с беззнаковыми целыми восьмеричными числами использует для представления числа динамический массив типа char, каждый из элементов которого содержит восьмеричную цифру. Младшая цифра имеет меньший индекс. Перегрузить арифметические операции (+, -, *, /) аналогичные встроенным для целых чисел, и операции сравнения (==, !=, <, >).

11.
Создать класс MString как производный от класса Array (динамический массив). Класс MString – для работы со строками (строка представляется как динамический массив байт, длина – в первом байте). Максимальный размер строки должен задаваться. Должны быть реализованы: определение длины строки, поиск подстроки в строке, удаление подстроки из строки, вставка подстроки в строку, перегруженная операция сцепления двух строк.

12.
Реализовать шаблон стека в виде растущего массива, т. е. массива с динамическим выделением памяти и возможностью перераспределения памяти, если не хватает.
Перегрузить операции ввода/вывода. Продемонстрировать не менее двух инстанцирований шаблона с разными аргументами и вызов всех методов.

13.
Реализовать шаблон «гибкого» массива, т. е. массива с возможностью вставки и удаления элементов со сдвигом остальных элементов, на основе динамического растущего массива, т. е. массива с динамическим выделением памяти и возможностью перераспределения памяти, если не хватает. Добавление и удаление элементов выполнять только по одному в произвольном месте.
Перегрузить операции ввода/вывода. Продемонстрировать не менее двух инстанцирований шаблона с разными аргументами и вызов всех методов.

14.
Реализовать шаблон числового «гибкого» массива, т. е. массива с возможностью вставки и удаления элементов со сдвигом остальных элементов, на основе динамического растущего массива, т. е. массива с динамическим выделением памяти и возможностью перераспределения памяти, если не хватает. Реализовать арифметические операции (+ - объединение, * - пересечение, * - умножение всех элементов на число, + - добавление числа ко всем элементам) и операции проверки на равенство и неравенство.
Перегрузить операции ввода/вывода. Продемонстрировать не менее двух инстанцирований шаблона с разными аргументами и вызов всех методов.

Добавление и удаление элементов выполнять только по одному; добавление – в конце массива, удаление – в начале.
Перегрузить операции ввода/вывода. Продемонстрировать не менее двух инстанцирований шаблона с разными аргументами и вызов всех методов.

Реализовать шаблон двусвязного списка. Реализовать операции доступа по индексу, + - сцепление двух списков, ==, !=, =, конструктор копирования, деструктор.
